# ICU Billing Denial Prediction Using Machine Learning

## Project Overview

This project focuses on predicting ICU billing denials associated with non-indicated ICU stays using clinical and administrative data.

The objective is to explore how machine learning can help identify patterns related to billing denials and support decision-making in healthcare operations and revenue cycle management.

## Current Stage

This repository currently includes the initial exploratory data analysis (EDA) notebook.

## Project Structure

- `01_eda_icu_billing_denials.ipynb`: exploratory data analysis of the dataset
- `Xgboost.ipynb`: data preprocessing
- `model_smote_SD.ipynb`: model training, model evaluation and shap interpretation
